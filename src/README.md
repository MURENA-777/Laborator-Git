# Лабораторная работа №10 - Управление проектом на GitHub

## Описание проекта
Проект демонстрирует использование Git и GitHub для управления разработкой.

## Структура проекта
- `src/main.py` - основной файл программы
- `src/feature.py` - дополнительный функционал
- `tests/` - директория с тестами

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`

## Использование
Запуск: `python src/main.py`
