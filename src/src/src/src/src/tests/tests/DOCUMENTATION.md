# Документация проекта

## Команды Git использованные в лабораторной:
1. `git init` - инициализация репозитория
2. `git clone` - клонирование репозитория
3. `git add` - добавление файлов
4. `git commit` - создание коммита
5. `git push` - отправка на сервер
6. `git pull` - получение изменений
7. `git merge` - слияние веток
8. `git branch` - работа с ветками
9. `git tag` - создание тегов
